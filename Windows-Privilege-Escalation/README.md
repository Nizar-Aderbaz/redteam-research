## What I Show in the Video
In the video, I walk through many real techniques step by step, including:

- Using **PowerUp** and **PrivescCheck** to find possible privilege escalation paths.
- Searching for **files that contain passwords**.
- Using **Unattended Installation files** to get stored credentials.
- Checking **Windows Credential Manager** for saved logins.
- Testing a **basic password-stealer** for learning purposes.
- Reading **PowerShell history** to find commands or secrets.
- Exploiting a **modifiable service executable**.
- Abusing **missing or broken scheduled tasks**.
- Using the **AlwaysInstallElevated** vulnerability to gain higher privileges.

## Goal of This Project
The goal is to help beginners understand Windows privilege escalation without complicated explanations.  
Everything is shown step by step in a safe lab environment.
